# HealthOnTheGo 🏥

A mobile health companion app for users in Nigeria and West Africa.
Find nearby clinics, get AI-powered health guidance, access emergency contacts, and more.

## 🔗 Live App
[HealthOnTheGo]([https://your-vercel-url.vercel.app](https://health-on-the-go-cr16.vercel.app/))

## Features
- 🗺️ Clinic Finder
- 💡 Health Tips
- 🔍 AI Symptom Guide
- 📹 Teleconsultation
- 📅 Appointment Scheduling
- 🤖 Health AI Chat
- 🚨 Emergency Contacts

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Node.js (backend, coming soon)
- Vercel (deployment)
- GitHub (version control)
